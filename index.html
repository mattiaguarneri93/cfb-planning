🔄 Sincronizzazione Google Sheets

Sincronizza i dati tra più dispositivi (telefono + PC) usando un Google Sheet come backend gratuito.

Mai sincronizzato
📖 Come configurarla (istruzioni)
Setup in 5 minuti (una tantum):
  1. Vai su sheets.google.com → crea un nuovo foglio vuoto
  2. Menu Estensioni → Apps Script
  3. Cancella il codice presente e incolla:
    function doGet(e){
  var s=SpreadsheetApp.getActiveSheet();
  return ContentService.createTextOutput(s.getRange('A1').getValue()||'{}')
    .setMimeType(ContentService.MimeType.JSON);
}
function doPost(e){
  SpreadsheetApp.getActiveSheet().getRange('A1').setValue(e.postData.contents);
  return ContentService.createTextOutput('OK');
}
  4. Salva (💾), poi Distribuisci → Nuova distribuzione
  5. Tipo: App web. Esegui come: Me. Chi ha accesso: Chiunque
  6. Autorizza, copia l'URL che finisce con /exec
  7. Incollalo qui sopra, spunta la sync automatica, clicca Salva poi Push
  8. Sugli altri dispositivi (telefono, altro PC): apri il file, incolla lo stesso URL, clicca Pull → sarai sincronizzato ✅
Chiunque abbia l'URL può leggere/scrivere. Non condividerlo pubblicamente.
